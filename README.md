# Movies
## :pencil: Project Description
Web Application for movies type Imdb build with Express.js, Handlebars view engine and MongoDb Database.

## :hammer: Used technologies
* HTML5
* CSS3
* Express.js
* MongoDB (Mongoose)
* Handlebars
